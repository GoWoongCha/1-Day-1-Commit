#클래스와 객체의 개념
클래스: 객체를 만들기 위한 설계도
객체: 설계도로부터 만들어낸 제품

속성 - 체력, 공격력, 방어력, 이동속도 #어떠한 특징을 나타냄
메서드 - 위치로 이동하기, 공격하기, 아이템 사용하기, 귀환하기 #어떠한 동작(함수)을 나타냄

#클래스 만들기
[만들기]
class 클래스이름:
	def 메서드 이름(self):
		명령블록

class Monster:
	def say(self):
		print("나는 몬스터다")

[호출하기]
인스턴스 = 클래스이름()
인스턴스.메서드()

goblin = Monster()
