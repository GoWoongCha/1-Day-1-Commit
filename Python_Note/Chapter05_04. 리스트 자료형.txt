#학습목표
리스트를 사용하는 이유
리스트 만드는 방법
데이터 접근하기
데이터 추가, 수정, 삭제

리스트 길이, 리스트 정렬

#리스트 사용하는 이유
EX) 10개의 동물이름 데이터를 저장

animal1="사자"
animal2="호랑이"
animal3="고양이
...
animal10="강아지"

=>만약 이름의 갯수가 아주 큰 숫자라면 일일이 변수를 설정하는 것이 번거로워진다.

#리스트
EX) 리스트 활용 방법

리스트명 = [데이터, 데이터, ... , 데이터]
animals=["사자", "호랑이", "고양이", ... , "강아지"]

리스트명 = [] => 빈 리스트 만들기

#데이터 접근하기
EX) 기차 비유

animals=["사자", "호랑이", "고양이", ... , "강아지"]

"사자" = animals[0]
"고양이" = animals[2]

#데이터 조작하기 

데이터 추가: 리스트.append(데이터)
데이터 할당: 리스트[인덱스]=데이터
데이터 삭제: del 리스트[인덱스]

슬라이싱: 리스트[시작:끝+1]
리스트 길이: len(리스트)
리스트 정렬: 리스트.sort()
