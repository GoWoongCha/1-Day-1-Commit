#제어문을 사용하는 이유
프로그램은 기본적으로 위에서 아래로 순차적으로 실행

명령 A, B 중 한 개를 선택하여 실행하고 싶거나 - 조건문
명령들을 반복해서 실행하고 싶을 때 - 반복문

치킨 or 피자? - 조건문
유튜브 영상보기 - 반복문

#조건문의 개념<순서도>

#if문 사용법
기존 비밀번호 = "1234"
입력한 비밀번호 = "1234"
만약 비밀번호를 정확히 입력했으면 -> 로그인 성공

origin_pass = "1234"
input_pass = input("비밀번호를 입력하세요 >>> ")

if input_pass == origin_pass: #조건 A
	#조건 A가 참이라면
	print("로그인 성공!")
	print("반갑습니다.")
elif input_pass == "": #조건 B
	#조건 A가 거짓이고 조건 B가 참이라면
	print("비밀번호를 입력하지 않았습니다.")
else: #조건 A,B가 거짓이라면
	print("로그인 실패!")
	print("비밀번호를 확인해주세요")
